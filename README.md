<h2>CoffeeShop</h2>
<p>
  E-commerce сайт кофейни для заказа кофе, тортов, выпечки и тд. Стэк: Asp net core 6, EF Core, C#, HTML5, Bootstrap 5, CSS3, JavaScript, Ajax.
